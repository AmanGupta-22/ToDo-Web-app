# Todo App - Spring Boot Project

A simple **To-Do Application** built with **Spring Boot**, **Thymeleaf**, and **MySQL**.  
This project allows users to **add, update, delete, and toggle tasks** in a web interface.

---

## Features

- Add new tasks
- Update task titles
- Delete tasks
- Mark tasks as completed (toggle)
- Responsive UI with **Bootstrap**
- Server-side rendering with **Thymeleaf**

---

## Technologies Used

- **Java 21**  
- **Spring Boot** (MVC)  
- **Thymeleaf** (Template Engine)  
- **MySQL** (Database)  
- **Bootstrap 5** (Frontend styling)  
- **Maven** (Build tool)
